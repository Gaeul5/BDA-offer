# BDA-offer
2024 BDA 채용공모전
제주도 신재생 에너지 발전량 예측을 통한 전력 비용 절감 및 탄소 배출 감소 분석 프로젝트

**프로젝트 목표**: 태양광·풍력 발전량 예측을 통한 공급 과잉 해결 및 경제적/환경적 효과 산출

**데이터**: 기상청(기상), 한국전력(전력 사용량), 제주에너지공사(발전량), 전력통계정보시스템(단가)

**사용 모델**: Random Forest(최적), XGBoost, LSTM, GRU, ARIMA

**핵심 성과**: 발전량 예측 기반의 연료비 정산단가 절감액 및 탄소 배출 감소량 정량화
